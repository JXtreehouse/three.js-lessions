# Typescript Three.js Webpack Starter

- Typescript support.
- Webpack
- Html plugin
- mini-css-extract-plugin

## Usage

### Install Typscript

```
npm install typescript -g
```

### Start

```
$ git clone 
$ cd first-demo
$ npm install # or yarn
$ npm start
```
